# library-management-system
This Library Management System is a Python console project for managing books, issues, returns, and fines using a SQL database.
A Python console project to manage books, issue/return operations, and fines using a SQL database.
Functions
Add Book – Add new books with title, author, and quantity.
Remove Book – Delete books by ID.
Issue Book – Issue books to users with automatic return date.
Return Book – Return books and calculate fine if late.
View Books – List all books with details.
